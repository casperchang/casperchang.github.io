Programming, reading, photography, and trying to write something.

My name is Casper, a software developer and visualization enthusiast. I used to write codes for image processing, GUI, medical devices, art filters for digital camera, iOS App, and wearable device. Now working in Sinopac Securities Taiwan as an UI Enthusiast.

One day, I found that it's so interesting to combine data processing and visualization to tell stories.
So I create this website to collect my visual works, they are mainly made by D3.js, python matplotlib module, or Processing language.
Most data came from my web scrapers written with various python modules, then parsed and filtered them by python programs.
